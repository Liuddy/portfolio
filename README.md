# Liuddy's Portfolio

## Introduction
This project is just a personal portfolio website so I can collect and show my work online.

## Features
- Direct access or a preview of each of my notifiable works.

## Installation
Clone the repository and open `index.php` in your browser to start exploring my portfolio, or simply click on this [link](https://Liuddy.github.io/portfolio/).

```
git clone https://github.com/Liuddy/portfolio
```

## Structure and Components
- `index.php`: Entry point that sets up the HTML structure and provides access to the page for each work.
